About this website

This website is built on the Angular v17 development platform. Angular is a TypeScript-based, open-source, single-page web application framework. This website is also:

  • built with Material Design, the official Angular Material UI components library.;
  • tested with the Jasmine testing framework and the Karma test runner. The UI components are tested with Angular Material's component test harnesses. The test code coverage is:
  • published with the Azure Static Web Apps service.
  • open-source in the my-website GitHub repository.
  • continuosuly integratated with GitHub Actions to manage the build, testing and deployment processes.
